What is box Model

The CSS box model is a container, which wraps around every HTML element. It consists of margins, borders, padding along with the actual content.

The space from the content to the border is padding. The green box around the element is the border and the thickness of the border is the width of the border. The spacing from the HTML element to the neighboring elements is the margin. The width and height of an element include padding, border, and margin

Border Radius

Border property adds a sharp border to our HTML element.

If we need rounded corners for our border, we can achieve the requirement using the border-radius property of CSS3. The border-radius is a shorthand property. Border radius value can be expressed using px or %. Percentage(%) is used when we want border-radius to be directly correlated with elements width. border-radius is a combination of below four properties. border-top-left: can be used to provide top left radius border-top-right: can be used to provide top right radius border-bottom-left: can be used to provide bottom left radius border-bottom-right: can be used to provide bottom right radius border: lightgrey dashed 3px;

Margin

Margin is also a shorthand property like padding. We can provide margin-top, margin-right, margin-bottom,margin-left values

Total Width and Height of an HTML element

When we specify the width and height property for any HTML element using CSS, we are actually setting the width and height of the content not the width and height of the complete element. The complete width and height of an element include the padding, border Total element width = width + left padding + right padding + left border + right border Total element height = height + top padding + bottom padding + top border + bottom border

Display Property

The display property specifies how the element should be rendered. The most common values provided to the display property are inline, block, inline-block, flex, and grid inline: The HTML element applied with this property will be rendered inline. We cannot apply height and width properties. block: The HTML element applied with this property will be rendered as a block element in the next line inline-block: The HTML element will behave like an inline element but we can apply width and height properties
