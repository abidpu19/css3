CSS3 allows us to add shadow to any element in the web page using the box-shadow property. 

        box-shadow:5px 10px 6px 2px rgba(30,50,40,0.5);

In the above example :

  • 5px indicates horizontal offset
  • 10px indicates Vertical offset
  • 6px indicates Blur Distance
  • 2px indicates Spread Distance
  • rgba(30,50,40,0.5) is used to apply shadow color

Let us have a glance over these properties and understand how to write with them.

Box Shadow properties
Property Description
horizontal offset Positive value creates a shadow to the right. Negative value creates the shadow to the left
vertical offset Positive value creates a shadow to the bottom Negative value creates the shadow to the top.
spread distance Optional value. Positive value expands the shadow in all directions. Negative value contracts the shadow.
blur distance An optional value that decides the percentage of blur effect takes only a positive value.
color An optional value that specifies the shadow color.
