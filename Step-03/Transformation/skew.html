CSS3 skew() function is used to turn/skew any element to any angle.

The turning/skewing is dependent on the parameters specified for the vertical & horizontal axis.

Syntax: skew(x,y) -> skews the element from x degrees from the x-axis & y- degrees from the y-axis. if the y-axis value is not specified then it has a zero value. To skew an element across a single axis, we can use skewX() and skewY() functions.

Example

  1. skew(20deg) -> skew an element by 20 degrees across x-axis.
  2. skew(30deg, 10deg) -> skews an element by 30 degrees across x-axis and 10 degrees across y-axis.
  3. skewX(15deg) -> skews an element by 15 degrees across x-axis.
  4. skewY(45deg) -> skews an element by 45 degrees across y-axis.
