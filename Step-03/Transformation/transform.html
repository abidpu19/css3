Why Transformation

You might have observed an HTML element getting transformed such as the element gets rotated or scaled upon hover of the mouse. These transformations were generally achieved through JavaScript or JavaScript frameworks. CSS3 has introduced the transformations using which we can transform our HTML elements using the transform property of CSS3. Transformations are applied to create dynamic web pages.

Translate

CSS3 translate() function is used to move an element from one position to another position An element can be moved to the right, left, top, bottom.

Syntax: translate (x,y) -> moves the element to right by x-value, and y-value down from its current position. An element can be moved in only one direction i.e. horizontal/vertical using translateX and translateY functions
