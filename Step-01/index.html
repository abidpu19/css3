What are Selectors ?

CSS selectors can be related to the previously discussed example scenario. Sometimes we might have a requirement to style all the paragraph elements of a web page, we would select all the paragraph elements in order to style them, and this is similar to selecting the mobiles out of all the gadgets. Sometimes we might want to style only a set of paragraphs out of all the paragraphs in a web page and this is similar to selecting a specific brand and model. Sometimes we might want to style only one paragraph, and this is similar to selecting a specific color from different colors available. There are different types of selectors available in CSS

  • Element Selector
  • Id Selector
  • Class Selector
  • Contextual Selectors
  • Grouping of selectors

CSS pseudo-class is a keyword that can be applied to selectors. Pseudo-classes specifies the state of the element to be selected

Link related pseudoclasses

The link is highlighted when hovered. This is achieved using :hover pseudoclass

The visited links appear in purple color. This is achieved using :visited pseudo class

Input related pseudo-classes

The Last Name is an enabled text field, which is applied with light blue background-color using :enabled pseudoclass. The Last Name field is a mandatory field, which is applied with red background-color using :required pseudoclass.

The first letter of the paragraph is rendered in bigger font-size and purple color using the :first-letter pseudo-class
