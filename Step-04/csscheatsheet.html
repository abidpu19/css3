The below table that lists all the CSS3 relational & attribute selectors supported by HTML5 tags, their descriptions with an example for each property

CSS3 relational & attribute selectors
Property Description Example
A>B Selects any element B that is a direct child of A(ignores the deeper nested elements) section > p
A+B Selects the B element that is placed immediately after the A element and shares the same parent. This selects only the immediate sibling. h1 + p
A ~ B Selects all the B elements that are placed after the A element and share the same parent. This selects all the siblings. div ~ p
a [attribute] Selects all A elements with the specified attribute. a[target]{ background-color: green;}
a[attribute=value] Selects all A elements with specified attribute and value. a[target="_blank"]{ background-color: green;}
a[attribute^=value] Selects all A elements where the value of attribute starts with the specified value. a[title^="A1"]{ background-color: green;}
a[attribute$=value] Selects all A elements where the value of attribute ends with the specified value. [title$="tion"]{ background-color: green;}
a[attribute*=value] Selects all A elements where the value of the attribute contains the specified value. [title*="A1001"]{ background-color: green;}
